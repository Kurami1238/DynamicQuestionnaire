此站 DynamicQuestionnaire 為動態問卷管理系統
可自定義問卷並供人填寫。

資料庫為：MSSQL
備份檔：DynamicQuestionnaire.bak
從Index頁進入

FrontEnd-List： 問卷搜尋前台，可供搜尋問卷及前往問卷
FrontEnd-Form： 填問卷頁，*為必填
FrontEnd-ConfirmPage： 填完問卷後的確認頁，可反悔修改 
FrontEnd-Stastic： 統計頁，支援圖表顯示

RearEnd-List： 問卷管理搜尋系統，可搜尋、新增刪除問卷
RearEnd-Detail： 問卷設計頁，可自定義選項及問題種類
注意事項:
1.要從此頁回到List需至"問卷"頁籤按取消
2."問卷"頁籤的送出只會紀錄資料至Session，問題的送出才會存至DB
RearEnd-YokuaruMondai： 常用問題管理，可自定義，並在問卷設計時能套入
